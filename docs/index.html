功能介绍

扫码登录

使用 B站 App 扫描二维码即可快速登录，无需手动输入账号密码。支持多人账号轮流使用，登录状态安全持久。

多账号管理

同时管理多个 B站账号，一键切换当前活跃账号。支持查看账号基本信息（用户名、UID），方便多账号场景下的内容运营。

  • 多账号快速切换
  • 账号添加与删除
  • 活跃账号实时标识

智能自动回复

支持对视频评论、私信、粉丝关注三种来源自动回复。可自定义回复内容，支持 {用户名}、{时间} 变量，提供灵活的回复配置。

  • 自定义回复内容（支持变量）
  • 可配置回复来源（评论 / 私信 / 关注）
  • 间隔时间控制，避免频繁发送
  • 每个用户只回复一次选项
  • 回复记录查看与管理
